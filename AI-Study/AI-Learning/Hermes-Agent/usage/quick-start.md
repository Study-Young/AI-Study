# 快速上手

本文档帮助您快速上手 Hermes Agent，从安装到第一次对话只需几分钟。

## 前提条件

- 已按照对应平台的[安装指南](../README.md#目录)完成安装
- 已完成 `hermes setup` 配置
- 已配置至少一个 LLM 提供商的 API 密钥

## 第一步：启动聊天

打开终端，运行：

```bash
hermes chat
```

系统会加载配置并启动交互式对话界面。您将看到类似以下的提示：

```
╭─ Hermes Agent ───────────────────────────╮
│  模型: gpt-4o                             │
│  平台: terminal                           │
│  技能: 已加载 3 个技能                    │
╰───────────────────────────────────────────╯

You >
```

## 第二步：开始对话

在 `You >` 提示符后输入您的消息，按 Enter 发送：

```
You > 你好！请介绍一下你自己。
```

AI 会回复一条消息，然后等待您的下一条输入。

```
You > 帮我写一个 Python 脚本来计算斐波那契数列。
```

## 第三步：使用斜杠命令

在聊天中输入斜杠命令来控制会话：

| 命令 | 说明 | 示例 |
|------|------|------|
| `/new` | 开始新对话 | `/new` |
| `/retry` | 重新生成上一条回复 | `/retry` |
| `/model` | 切换模型 | `/model claude-sonnet-4-20250514` |
| `/exit` | 退出聊天 | `/exit` |
| `/help` | 显示帮助 | `/help` |

例如，要切换模型：

```
You > /model claude-sonnet-4-20250514
```

## 第四步：使用技能

Hermes Agent 可以加载技能来执行特定任务。使用 `/skills` 命令查看可用技能：

```
You > /skills
```

如果启用了搜索技能，可以直接让 AI 搜索互联网：

```
You > 使用搜索技能查询今天的新闻
```

## 第五步：切换平台

Hermes Agent 支持在多个平台上运行。配置完成后，可以在不同平台之间切换。

查看当前平台：

```
You > /status
```

## 实用技巧

### 多行输入

需要输入多行文本时，使用 `\` 结尾：

```
You > 请帮我写一个详细的计划，\
包含以下几个部分：\
1. 目标\
2. 步骤\
3. 时间线
```

### 文件引用

可以引用本地文件作为对话上下文（需要启用文件技能）：

```
You > 请分析这个文件：/home/user/data.txt
```

### 中断生成

如果 AI 正在生成回复，按 `Ctrl+C` 中断。

### 查看上下文使用情况

```
You > /status
```
