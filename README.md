# AI-Study

> AI 学习与资讯项目 — 系统化学习当下最热门的 AI 工程工具

---

## 项目结构

```
AI-Study/
├── AI-Learning/         # AI 学习模块
│   ├── Claude-Code/     # Anthropic Claude Code — CLI 编码代理
│   ├── Hermes-Agent/    # Nous Research Hermes Agent — 全功能 AI Agent 框架
│   └── OpenCode/        # OpenAI Codex CLI — 开源 AI 编码代理
├── AI-News/             # AI 资讯模块 — 定期总结 AI 技术动态
└── Requirements/        # 需求与计划文档
```

## 模块介绍

### 📚 AI 学习模块

三大主流 AI 工程工具的完整学习资料，覆盖 **安装部署 → 使用方法 → 进阶功能 → 实战示例** 全链路。

| 工具 | 类型 | 开源 | 核心用途 |
|------|------|------|---------|
| [Claude Code](./AI-Learning/Claude-Code/) | 编码代理 | ✗ (Anthropic) | 代码编写、审查、重构、调试 |
| [Hermes Agent](./AI-Learning/Hermes-Agent/) | Agent 框架 | ✓ (Nous Research) | 自动化工作流、多平台 Agent、定时任务 |
| [OpenCode](./AI-Learning/OpenCode/) | 编码代理 | ✓ (MIT) | 代码生成、PR 审查、多代理协作 |

### 📰 AI 资讯模块

定期从主流 AI 信息源采集并提炼技术动态，归档为结构化摘要文档。

- 如果你是**初学者** — 从各工具的 `installation/` 开始
